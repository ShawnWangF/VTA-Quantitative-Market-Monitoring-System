# VTA / F-LOAM 研究记录

## 当前已核验结论

| 主题 | 结论 | 对当前系统的意义 |
| --- | --- | --- |
| F-LOAM | F-LOAM 是一篇 **2025 年**股票预测论文提出的混合模型，核心并不是新闻理解型 LLM，而是 **SVMD 去噪 + LSTM 特征抽取 + LightGBM 预测 + Optuna 超参优化**。 | 适合借鉴为**量化执行层**与**回测优化层**，不应被误当作完整的宏观语境理解模型。 |
| 多模态金融代理 | FinAgent 一类工作更接近“**新闻/文本/图像/数值时间序列融合 + 记忆/反思**”路线。 | 适合借鉴为**语境理解层、解释层与学习反馈层**。 |
| 当前缺口 | 现有项目中的 `buildSignalBlueprint` 主要仍是基于价量阈值的启发式规则，尚未形成真正的“事件语境输入层”。 | 需要新增事件分数、情绪权重、宏观冲击和反思更新机制。 |

## F-LOAM 摘要要点

> F-LOAM 的论文标题为“F-LOAM: an efficient hybrid model for stock price prediction based on SVMD denoising”。文中摘要显示，该模型强调通过 **SVMD 去噪** 改善金融时间序列质量，再用 **LSTM + LightGBM** 的级联架构预测价格，并通过 **Optuna** 做参数优化，最终再将结果转换为可执行交易策略。

这说明 **F-LOAM 更像一个预测与执行优化框架**，并不天然负责新闻、社交媒体、财报、宏观指标的语义理解。

## 多模态金融代理要点

> FinAgent 这类论文强调一个多模态市场情报模块，可以把 **新闻、情绪、视觉图表和数值时间序列** 联合处理，并结合记忆检索与反思模块来改进交易决策。

这条路线更接近用户希望的：由大模型理解宏观语境和事件，再把结果交给量化模块做 buy / sell 点位执行。

## 当前初步映射建议

| 目标层 | 推荐参考 | 说明 |
| --- | --- | --- |
| 语境理解层 | 多模态金融代理 / LLM 事件理解路线 | 负责把新闻、社交情绪、财报、宏观信息转成结构化事件分数。 |
| 量化执行层 | F-LOAM 式时序预测与优化思想 | 负责把事件分数和实时价量特征合并，输出具体触发位、止损位和失效条件。 |
| 学习反馈层 | 反思模块 + 强化式权重更新 | 根据模拟交易结果调整事件权重、阈值和信号可信度。 |

## 待继续核验问题

1. “VTA” 在用户所指的金融交易语境中，是否有一个明确且权威的公开论文或标准全称。
2. 是否存在比 FinAgent 更接近用户所说“VTA”语义的模型或框架。
3. 当前项目中可接入的外部数据源和频率，是否足以支撑新闻/情绪/宏观事件评分层。

## 第二轮核验补充

| 主题 | 新结论 | 对当前系统的意义 |
| --- | --- | --- |
| VTA | 在目前能核验到的金融论文里，VTA 更明确对应 **Verbal Technical Analysis**，而不是此前假设的 Visual-Temporal Alignment。它强调把价格时间序列转成文本注释，再用 LLM 进行可解释推理，并结合时序骨干模型输出预测轨迹。 | 适合借鉴为**图上解释层**与**可解释预测层**，特别适合把 BUY / SELL 点解释成“由哪些技术结构与时间序列特征触发”。 |
| FLAG-Trader | FLAG-Trader 提供了更直接的 **LLM + 强化学习** 金融交易框架：用策略梯度 / actor-critic 思想，根据 PnL 和 Sharpe 风格奖励在线更新策略。 | 适合借鉴为**实时回测强化学习回路**，但当前系统宜先做轻量化版本：以模拟交易收益、回撤、失效原因驱动权重更新，而不是直接在线重训大模型。 |
| 名称校正 | 用户提到的“VTA + F-LOAM + RL”组合，若严格按公开文献理解，更合理的落地方式应是：**VTA 负责可解释时间序列推理，F-LOAM 负责预测与执行优化，FLAG-Trader 风格模块负责强化反馈。** | 当前系统可以做成混合架构，但不应声称完全复现一个单一统一论文体系。 |

## 架构修正后的初步方案

| 层级 | 建议来源 | 当前落地方式 |
| --- | --- | --- |
| 时间序列解释层 | VTA | 将两支股票的最新分时、技术结构、触发状态转成结构化文本，由服务端大模型生成可解释推理链与预测偏向。 |
| 预测执行层 | F-LOAM | 用实时价量、事件分数、波动特征、历史触发结果合成触发价、止损位、失效条件与信号置信度。 |
| 强化反馈层 | FLAG-Trader | 用模拟交易的收益、Sharpe 风格分数、回撤惩罚、失效原因更新事件权重与阈值。 |

## 当前研究阶段的关键判断

目前可以较有把握地说：

1. **F-LOAM 可直接作为量化预测执行层的参考。**
2. **VTA 可作为可解释时序推理层的参考，但它更偏技术分析解释，不等于宏观新闻理解本身。**
3. **若要实现“宏观语境理解 + 强化学习自演化”，还需要额外补入事件语境输入层和 RL 奖励更新层。**
4. 对于当前项目，更现实的实现路径是做一个 **VTA + F-LOAM + FLAG-Trader 风格反馈** 的轻量混合版，而不是宣称完全原样复刻单篇论文。
